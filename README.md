# 🏥 Medicare Pro Dashboard

A comprehensive Hospital Management System with dedicated portals for both Staff and Patients. Built using the MERN stack (MongoDB, Express, React, Node.js) with secure JWT authentication.

## ✨ Key Features

### 👨‍⚕️ Staff Portal
* **Real-time Analytics:** Summary cards for total revenue, patient count, and active appointments.
* **Management Suites:** Full CRUD operations for Patients, Doctors, Appointments, Billing, Pharmacy, and Lab Tests.
* **Professional UI:** Includes instant search filtering, pagination, and status-based color coding.

### 👤 Patient Portal
* **Self-Registration:** Patients can create their own accounts.
* **Personalized Dashboard:** Patients only see their own appointment history.
* **Booking System:** Easy-to-use form to request new appointments with specific doctors.

## 🛠️ Tech Stack
* **Frontend:** React.js, Bootstrap 5, Axios
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Security:** JSON Web Tokens (JWT) for Role-Based Access Control
